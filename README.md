## 🚀 Projetos em Destaque

### 📈 Análise de Churn Predict
Análise detalhada para identificação de padrões de cancelamento de clientes.
* **Problema:** Alta taxa de evasão em serviços de assinatura.
* **Solução:** Pipeline em Python para limpeza e análise exploratória (EDA).
* **Resultado:** Redução teórica de 56% para 18% na taxa de cancelamento através de estratégias baseadas em dados.

### 🤖 Automação de Registros ERP
Automação de tarefas repetitivas de cadastro para otimização de tempo.
* **Ferramentas:** Python, Pandas, PyAutoGUI.
* **Ganho:** 60% de aumento na eficiência operacional da equipe administrativa.
